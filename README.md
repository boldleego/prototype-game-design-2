# Prototype Game Design 2

Godot prototype for a spherical puzzle-action game inspired by Magical Drop.

## Current Prototype Focus

- Fixed camera around a central sphere
- Player movement as a cursor moving over the globe
- Rotatable movement frame relative to camera orientation
- Mechanics-first design planning for a spherical tile board

## Key Files

- `project.godot`
- `scenes/main.tscn`
- `scripts/player_ship.gd`
- `PROTOTYPE_SPEC.md`
